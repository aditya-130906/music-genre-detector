                                          Music Genre Classification using PyTorch

This project implements a deep learning model to classify music into genres using audio signals. The model is trained on the GTZAN dataset and uses Convolutional Neural Networks (CNNs) on Mel spectrograms.






                                                       Model Architecture

3 Convolutional Layers (Conv2D)

ReLU Activation

Max Pooling

Adaptive Average Pooling

Fully Connected Layers

Dropout for regularization


                                                              Dataset

This project uses the GTZAN Genre Classification Dataset.

1000 audio files

10 genres (blues, jazz, rock, etc.)

Each clip is 30 seconds long
